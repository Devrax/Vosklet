Vosklet local · reconocimiento de voz sin conexión

Ejemplos

Cuatro variantes del mismo reto de voz en español. Todo el reconocimiento ocurre en tu dispositivo.

Reto de voz

Motor en el hilo principal (monosklet/singlethread): graba una frase, transcríbela y compárala con el reto escrito.

Reto de voz · Web Worker

El mismo reto con el motor dentro de un Web Worker (monosklet/worker): la interfaz nunca se bloquea durante la transcripción.

Reto de voz · Verificación de hablante

Añade verificación de voz (speaklet): registra tu voz de referencia y comprueba quién habla al superar el reto.

Reto de voz · Identificación x-vector

Identificación de hablante con el modelo nativo de Vosk (vosk-model-spk-0.4) dentro de un Web Worker (monosklet/worker): registra tu voz y compara huellas x-vector por similitud coseno.
